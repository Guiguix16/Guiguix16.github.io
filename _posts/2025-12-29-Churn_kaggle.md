## Description

Ce projet est basé sur un challenge Kaggle portant sur l’analyse d’un jeu de données réel.
L’objectif était de comprendre les données disponibles, de construire un modèle de prédiction
et d’évaluer les résultats obtenus.


---

## Le jeu de données

Le jeu de données fourni dans le cadre du challenge regroupe plusieurs variables descriptives
liées aux observations à analyser.
Comme c’est souvent le cas sur ce type de problématique, les données présentent
des valeurs manquantes, des distributions hétérogènes et des informations de nature différente.

Une étape importante du travail a consisté à :
- comprendre le rôle de chaque variable
- identifier les données incomplètes ou peu exploitables
- vérifier la cohérence globale du jeu de données
- 
- **Données Kaggle** :  [Lien du Challenge](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)

---

## Travail réalisé

Le projet s’est articulé autour des étapes suivantes :

- exploration des données afin d’identifier les tendances générales
- préparation des données pour les rendre exploitables
- sélection et transformation des variables jugées pertinentes
- entraînement et comparaison de plusieurs approches de modélisation
- analyse des résultats obtenus

L’objectif principal n’était pas de maximiser la performance,
mais de s’assurer que les résultats soient compréhensibles et cohérents
au regard des données disponibles.

---

## Points clés du projet

Certains aspects ont été particulièrement importants dans ce travail :

- prendre le temps de comprendre les données avant toute modélisation
- limiter la complexité inutile
- vérifier la stabilité des résultats
- documenter les choix effectués tout au long du projet

Ces points ont guidé l’ensemble du développement.
